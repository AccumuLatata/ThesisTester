# METRICS GLOSSARY

All metrics below are computed from the realized `r_multiple` series generated by the backtest engine.

## Core formulas

### R-multiple
\[
R_i = \frac{\text{pnl\_points}_i}{\text{sl\_pts}}
\quad\text{where}\quad
\text{sl\_pts} = \text{stop\_loss\_ticks} \times \text{tick\_size}
\]
Implementation: `sl_pts` and `r_multiple` in `thesistester/engine/backtest.py:131-132,260`.

### Win rate
\[
\text{win\_rate} = \frac{\#\{R_i > 0\}}{n}
\]
Implementation: wins filter and ratio in `thesistester/analytics/metrics.py:51,54`.

### Profit factor
\[
\text{profit\_factor} =
\begin{cases}
\frac{\sum_{R_i>0} R_i}{\left|\sum_{R_i<0} R_i\right|}, & \sum_{R_i<0} R_i < 0 \\
\infty, & \sum_{R_i<0} R_i = 0 \text{ and } \sum_{R_i>0} R_i > 0 \\
0, & \text{otherwise}
\end{cases}
\]
Implementation: `gross_win`, `gross_loss`, and branches in `thesistester/analytics/metrics.py:60-67`.

### Expectancy (R)
\[
\mathbb{E}[R] = \text{win\_rate} \cdot \overline{R}_{win} + \text{loss\_rate} \cdot \overline{R}_{loss}
\]
Fallback: when one side is missing (all-win/all-loss), code uses `avg_r`.
Implementation: `thesistester/analytics/metrics.py:72-76`.

### Drawdown with `clip(lower=0)` anchoring
\[
\text{cum\_r}_t = \sum_{i=1}^{t} R_i
\]
\[
\text{running\_max}_t = \max_{k \le t}(\text{cum\_r}_k)\ \text{clipped at}\ \ge 0
\]
\[
\text{drawdown}_t = \text{running\_max}_t - \text{cum\_r}_t
\]
\[
\text{max\_drawdown\_r} = \max_t(\text{drawdown}_t)
\]
Implementation: `cum_r`, `cummax().clip(lower=0.0)`, and drawdown in `thesistester/analytics/metrics.py:82-85` (and same anchor logic in equity curve at `thesistester/analytics/metrics.py:203-206`).

## Notes
- Metrics are gross because trade P&L is computed without commissions/fees/slippage (`thesistester/engine/backtest.py:73-82,259-260`).
- Backtest directional KPIs ("Long vs Short KPIs") use the same formulas above, applied independently to `direction == "long"` and `direction == "short"` subsets.
- Validation pages label outputs as diagnostic only (`pages/10_Validation.py:18`, `thesistester/analytics/validation.py:13`).
